Persistent Navigation and Back To Top Button

Persistent Navigation and Back To Top button is created using JavaScript and styled with CSS. It's a different style of navigation that appears when the user is scrolling through the page. With this style of navigation, the navigation bar is always available to the user instead of them having to scroll back up to the top. If the user does want to get back to the top the Back To Top button send them back to the top of the page. The $(window).scroll(function() is used to change the Persistent Navigation back to the Tab Style Navigaiton. It is also used for the Back To Top button. Once the user scrolls 20 pixels the function adds the class floater to the navigation and the Back To Top button which gives the navigation that Persistent Navigation look and once it gets back to the top it removes that class making it Tab Style again. The $('footer a.top').click(function(event) is similar to the Persistent Navigation function but it causes the Back To Top button to disappear once it reaches the top and reappear once the user scrolls 20 pixels again.
