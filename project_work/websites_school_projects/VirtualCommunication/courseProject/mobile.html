Responsive vs Adaptive

Responsive and Adaptive Web Design are both methods of controlling the way your website reacts to different screen sizes. Responsive is a fluid style of design that responds immediately to the size of your screen. Media queries are used to control how it reacts. For example, if you set media queries to max-width: 760 pixels, max-width: 960 pixels, max-width: 1080 pixels you can change the way the site looks until it reaches 760 pixels wide in size, then how it looks until 960 pixels, and so on. Adaptive is a style of design that is done by creating multiple CSS layouts for multiple versions of your site. For example, you would create a layout for mobile, tablets, desktops, and larger screens. Once the web page loads, it identifies what device or size it is and will use the appropriate CSS layout.

Media Queries

Below, I have shown the media queries that I used for this site. Between 960 pixels and 1450 pixels of width, I set the page width and the persistent navigation bar to a full 100% page width. 960 pixels of width and less will remain at 100% page width but I have also made it so each of the two column layouts becomes one column at 100% width as well. 1450 pixels and above will have the default stylings.
