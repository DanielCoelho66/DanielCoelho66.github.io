Semantic Elements

Prior to HTML5 web pages used a lot of div elements to show divisions between areas of content. With HTML5, new semantic elements were introduced. These provide easier, cleaner, and more accessible code. You should move away from div elements and move towards semantic elements. HTML5 Doctor is a great site to learn HTML5 elements, as it provides definitions and examples of each element.

Accessibility

Skip to - link

Accessibility on the web is becoming more important, and devices like screen readers are helping people who have a vision impairment to surf the web. Screen readers are controlled by keyboard commands and will read out a web page's content. A Skip to - link is unseen by average users but allows someone with a screen reader to skip to different sections of the page.

WAI-ARIA

WAI-ARIA is a set of attributes used to increase accessibility of a web page. These attributes provide additional information for screen readers to provide an overall better experience and accessibility of the site for everyone.There are different types of WAI-ARIA attributes such as: Landmark roles, states and properties, and live regions. The W3C are the ones who help set the standards for WAI-ARIA and accessibility on the web.

Meta Tags

Meta elements are located in the head section and are used to provide additional metadata about a website. This additional metadata is useful for search engines and search results. When someone enters a search, if it matches up with the metadata(or the content of the web page), it'll be displayed on the search results page.
