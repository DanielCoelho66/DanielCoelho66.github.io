Tab Style Navigation

Tab Style Navigation is a way of styling your navigation bar so they appear as tabs. By adding position left: 50% and position right: 50%; the navigation bar will always be centered. The margin-right: 0.25em is what sets the space between each tab.

Column Layout

Column layout is done by using a little bit of math and setting the widths of the sections. There's 100% of space to work with, so I set the widths of both these sections to 45%, which leaves a 10% piece of space in the middle. Depending on how many columns or the importance of the section, the widths can be any size as long it doesn't go over 100%.
